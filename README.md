# Trader Behavior vs Market Sentiment Analysis

## Objective
The goal of this project is to analyze how Bitcoin market sentiment (Fear vs Greed) affects trader behavior and performance on Hyperliquid.

This analysis focuses on:
- Comparing trader performance during Fear vs Greed periods
- Understanding changes in trading behavior
- Identifying different trader segments
- Providing actionable trading insights
